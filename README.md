# math_lang
pretty printing calculator language

needs a better name

## Example

```
mass = 10 kg

F_0 = 1000 N = ?
dt_{force} = 0.5 s

y_{vel} = F_0 / mass * dt_{force} = ?

g = 9.81 m/s^2

dt_{air} = 10 s

h = y_{vel} * dt_{air} - ((1/2) * g * dt_{air}^2) = ?
```
