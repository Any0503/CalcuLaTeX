pretty printing calculator language

Kind of like smart LaTeX

needs a better name

## Example

```
g = 9.81 m/s^2
mass = 10 kg
F_0 = 1000 N
dt_{force} = 0.1 s

y_{vel} = F_0 / mass * dt_{force} = ?

airtime = y_{vel} / g * 2 = ?
height_{max} = y_{vel} * airtime / 4 = ? cm
```
